6.1.1. “Bearer”OAuth访问令牌类型
=============================
- Type name：

  Bearer
- Additional Token Endpoint Response Parameters:

  (none)
- HTTP Authentication Scheme(s):

  Bearer
- Change controller：

  IETF
- Specification document(s):

  [RFC 6750][RFC6750]
